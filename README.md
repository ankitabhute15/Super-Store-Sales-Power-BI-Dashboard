# Super-Store-Sales-Power-BI-Dashboard

### Description
A dynamic and interactive Power BI dashboard designed to analyze Super Store sales performance, uncover growth opportunities, and support data-driven decision-making.

### Objective
Showcase end-to-end data analytics skills by transforming Super Store data into a dynamic Power BI dashboard that uncovers sales trends, performance drivers, and actionable insights for business growth.
