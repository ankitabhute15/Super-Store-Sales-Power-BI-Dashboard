# Super-Store-Sales-Power-BI-Dashboard

### Description
A dynamic and interactive Power BI dashboard designed to analyze Super Store sales performance, uncover growth opportunities, and support data-driven decision-making.

### Objective
Showcase end-to-end data analytics skills by transforming Super Store data into a dynamic Power BI dashboard that uncovers sales trends, performance drivers, and actionable insights for business growth.

### Dataset Overview
The dataset includes:
- Orders and Sales Details: Sales, Quantity, Discount, Profit
- Product Details: Category, Sub-category
- Customer Information: Customer name, Segment
- Geographic Data: Region, State, City
- Time Data: Order date, Ship date

### Key Features
- Interactive slicers for Category, Sub-Category, Region, State, Segment
- KPI cards for total sales, total profit, average discount, quantity sold
- Profitability analysis using stacked bar charts & matrices
- Product-wise comparison for better inventory planning
- Geo-map to visualize top-performing states
- Customer-level contribution analysis

### Questions(KPIs)
- What are the total Sales, Profit, Quantity sold, and Average Discount?
- Which Category and Sub-category generate the highest revenue?
- Which States contribute the most to Sales and Profit?
- Who are the most valuable customers?
- What is the monthly Sales Trend?
- Which payment modes are most used?
- Which shipping modes are most preferred?
- Which Segment performs best?

### Tools & Skills Used
- Power BI Desktop
- Power Query (Data Transformation)
- DAX (Data Modeling & KPIs)
- Raw dataset provided in CSV format

### Insights Delivered
- Identified top-performing and underperforming product categories
- Highlighted region-wise sales and profit variations
- Displayed monthly sales and profit trends
- Analyzed customer segment performance
- Compared shipping modes and payment mode contributions
- Showed state-level performance using map visualization
